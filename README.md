# catgraphy

## アプリケーション概要
猫の写真を投稿して共有するアプリケーションです
<img width="1436" alt="catgraphy" src="https://user-images.githubusercontent.com/56598400/68197186-268b8f80-fffd-11e9-9fbd-90b9b32a5e71.png">


## アプリケーション機能一覧
- ユーザー管理機能
- コンテンツ投稿機能
- コンテンツ一覧表示機能
- コンテンツ編集機能
- コンテンツ削除機能
- 画像ファイルアップロード機能
- 画像をリサイズしてサムネイルとして保存する機能
- DBテーブルのリレーション管理
- ページネーション機能
- 単体テスト
- 統合テスト



## 使用技術
- HTML
- CSS
- PHP 7.3
- Laravel 5.4
- Intervention Image
- Bootstarp
- MySQL
